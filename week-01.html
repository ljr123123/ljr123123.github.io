第 1 周：环境与工具链

阶段：phase-00-setup | 预计总时长：约 18h

本周目标

搭建可复现的 Python 研究环境，跑通第一个量化脚本，了解主流量化框架生态。

验收标准

  • [ ] requirements.txt 依赖全部安装成功
  • [ ] python code/phase-00-setup/hello_quant.py 无报错，生成测试图
  • [ ] 能用 pandas 画出一只股票的收盘价曲线
  • [ ] 学习日志 ≥ 5 篇，周总结 1 篇

每日安排

周一（≤2h）

时长 任务
0.5h 阅读 docs/learning-plan.md 阶段 0、code/phase-00-setup/README.md
1h 创建虚拟环境：python -m venv .venv，激活并 pip install -r requirements.txt
0.5h 写今日日志，记录 Python 版本与 pip 输出

周二（≤2h）

时长 任务
1h 运行 hello_quant.py，排查 import 或路径问题
0.5h 阅读 pandas 10 minutes to pandas 前半
0.5h 日志：记录运行结果与环境信息

周三（≤2h）

时长 任务
1h 调研量化生态：backtrader、zipline、vnpy、qlib、vectorbt（各 10 分钟）
0.5h 写对比笔记：适用场景、活跃度、A 股支持
0.5h 日志

周四（≤2h）

时长 任务
1h 用 yfinance 或 akshare 拉取 1 只股票 1 年日频数据
0.5h 用 matplotlib 画收盘价折线图
0.5h 日志：附图表路径

周五（≤2h）

时长 任务
1h 练习 Git 基本流程：status → add → commit（本地即可）
0.5h 可选：配置 Jupyter / VS Code Python 解释器指向 .venv
0.5h 日志

周六（≤8h）

时长 任务
2h 完善 hello_quant.py：输出 numpy/pandas 版本、中文字体配置
2h 将周四绘图脚本整理为可复用函数，放入 code/phase-00-setup/
1h 对照 code/phase-00-setup/README.md 任务清单逐项勾选
1h 复习 pandas 10min 教程后半（GroupBy、时间序列）

周日（≤8h）

时长 任务
2h 撰写 learning-logs/YYYY-MM/week-01-summary.md
1h 自测：脱离文档能否从零建环境并跑通脚本
1h 预习第 2 周：浏览 code/phase-01-data/README.md、注册 tushare/akshare

推荐阅读

  • QuantStart 入门系列 — 选读 1–2 篇
  • pandas 官方 10 minutes to pandas
