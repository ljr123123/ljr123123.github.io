第 2 周：市场结构与数据获取（上）

阶段：phase-01-data | 预计总时长：约 20h

本周目标

理解 A 股低频交易的市场规则，选定数据源，完成行情下载脚本骨架。

验收标准

  • [ ] 学习日志 1 篇：A 股 T+1、涨跌停、ST、停牌规则
  • [ ] download_ohlcv.py 能下载 ≥3 只股票 ≥1 年日频 OHLCV
  • [ ] 数据落盘至 data/phase-01/raw/

每日安排

周一（≤2h）

时长 任务
1h 阅读 A 股交易规则：T+1、涨跌停板（10%/20%/5%）、ST 标识、集合竞价
0.5h 笔记：这些规则对回测成交假设的影响
0.5h 日志

周二（≤2h）

时长 任务
1h 对比 akshare / tushare / yfinance：免费额度、字段、延迟、A 股覆盖
0.5h 选定主数据源，完成 API token 或安装配置
0.5h 日志：记录选型理由

周三（≤2h）

时长 任务
1h 理解 OHLCV 字段：open/high/low/close/volume/amount，换手率
0.5h 试验下载单只股票，打印 DataFrame 结构
0.5h 日志

周四（≤2h）

时长 任务
1.5h 创建 code/phase-01-data/download_ohlcv.py：单票下载函数
0.5h 日志

周五（≤2h）

时长 任务
1.5h 扩展：支持股票列表批量下载、异常重试、进度打印
0.5h 日志

周六（≤8h）

时长 任务
3h 下载沪深 300 成分股（或自选 30 只）2015–今 raw 数据
2h 数据完整性检查：日期连续性、空值比例、价格异常值
1h 为 code/phase-01-data/README.md 补充实际运行命令
1h 阅读：前复权 vs 后复权概念（为下周准备）

周日（≤8h）

时长 任务
2h 阅读指数成分股历史变动与幸存者偏差（用中证指数公司资料或论文摘要）
2h 整理 data/phase-01/ 目录结构与 .gitignore 确认
2h 周总结 + 列出下周复权清洗待办

核心概念

  • T+1：当日买入次日才能卖出，回测调仓假设须匹配
  • 幸存者偏差：用「当前成分股」回测历史会高估收益
  • 交易日历：A 股节假日与自然日不同
